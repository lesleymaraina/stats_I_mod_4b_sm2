Review: Regression Test Statistic

Hypothesis tests are constructed using a regression test statistic:

\[ \frac{\text{estimate} - \text{hypothesized value}}{\text{standard error}} \]

  • estimate represents a regression coefficient while the standard error quantifies the expected sampling variability of that estimate

  • test statistic measures how far the observed coefficient is from the null hypothesis after accounting for uncertainty in the data

  • large absolute test statistic values provide stronger evidence against the null hypothesis and lead to smaller p-values

Current lecture will discuss the application and interpretation of these regression hypothesis tests within linear and logistic regression models

Evaluating Associations with Regression Models

Regression analysis is useful when the scientific question involves several patient, disease, or treatment characteristics simultaneously

Regression hypothesis testing helps answer whether the observed association is stronger than expected under the null hypothesis of no population association

Applied research question Outcome type Regression model
Is tumor stage associated with tumor biomarker level? Continuous Linear regression
Is mutation status associated with treatment response? Binary Logistic regression
How large is the estimated association? Continuous or binary Coefficient, confidence interval, or odds ratio

Review: Regression Hypothesis Test

For a single regression coefficient hypotheses include:

\[ H_0: \beta_j = 0 \]

\[ H_a: \beta_j \neq 0 \]

null hypothesis : states that the predictor has no association with the outcome

alternative hypothesis : states that the predictor has an association with the outcome

Review: Regression Test Statistic

A regression test statistic has the form:

\[ \text{test statistic} = \frac{\hat{\beta}_j - \beta_{j,0}}{SE(\hat{\beta}_j)} \]

  • regression test statistic: estimated coefficient minus the hypothesized coefficient value, divided by the standard error of the estimated coefficient

  • statistic measures how many standard errors the estimated coefficient is from the null value.

Decision Rule

Reject \(H_0\) if:

\[ P\text{-value} \leq \alpha \]

Fail to reject \(H_0\) if:

\[ P\text{-value} > \alpha \]

  • statistical significance : sample provides enough evidence to reject the null hypothesis at the selected significance level

Linear Regression Model

A multivariable linear regression model can be written as:

\[ E(Y \mid X_i) = \beta_0 + \beta_1X_1 + \beta_2X_2 + \beta_3X_3 + \beta_4X_4 \]

Formula interpretation: expected continuous outcome is modeled as a linear function of multiple predictors.

Where:

  • \(Y\) : tumor biomarker level
  • \(X_1\) : tumor stage
  • \(X_2\) : age
  • \(X_3\) : mutation status
  • \(X_4\) : baseline disease burden

Linear Regression Model

\[ E(Y \mid X_i) = \beta_0 + \beta_1X_1 + \beta_2X_2 + \beta_3X_3 + \beta_4X_4 \]

  • \(E(Y \mid X_i)\) : expected tumor biomarker level for a given set of predictor values
  • \(\beta_0\) : intercept
  • \(\beta_1\) : expected difference in tumor biomarker level for a one-unit increase in tumor stage, holding other variables constant
  • remaining coefficients interpreted while holding the other included variables constant

Linear Regression Hypothesis Test

  • test evaluates whether tumor stage is associated with expected tumor biomarker level after accounting for age, mutation status, and baseline disease burden

  • Biology Background: tumor stage describes how large a cancer is and how far it has spread in the body. Stages range from Stage 0, where abnormal cells are present but have not spread, to Stage IV, where cancer has spread to distant parts of the body, with Stages I–III representing increasing tumor size and/or spread to nearby tissues or lymph nodes

Tumor stage the coefficient of interest : \(\beta_{stage}\)

Null hypothesis :

\[ H_0: \beta_{stage} = 0 \]

Alternative hypothesis:

\[ H_a: \beta_{stage} \neq 0 \]

Logistic Regression: Practical Application

Logistic regression is used when the outcome variable has only two possible categories

Here we address:

After accounting for tumor stage, age, and baseline disease burden, is mutation status associated with higher or lower odds of treatment response?

In this study, the outcome is treatment response [response_label]:

1 : response

0 : no response

Note:

  • positive regression coefficients correspond to higher odds of response
  • negative regression coefficients correspond to lower odds of response
  • hypothesis tests and p-values evaluate whether the observed associations are likely to reflect true association

Logistic Regression Model

logistic regression formula states that the log-odds of the binary outcome are modeled as a linear function of the predictors:

\[ \log\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1X_1 + \beta_2X_2 + \beta_3X_3 + \beta_4X_4 \] Where:

  • \(p\) : probability of treatment response
  • \(1-p\) : probability of no response
  • \(\frac{p}{1-p}\) : odds of response
  • \(\log\left(\frac{p}{1-p}\right)\) : log-odds of response
  • \(\beta_1\) : change in log-odds associated with the predictor, holding other variables constant.

Note: log-odds are difficult to interpret directly therefore logistic regression coefficients are often exponentiated to produce odds ratios

Odds Ratio Interpretation

Exponentiating a logistic regression coefficient converts the estimate from the log-odds scale to the odds-ratio scale

\[ OR = e^{\hat{\beta}} \]

Example:

For mutation status, the odds ratio compares the odds of response for patients with mutation present to patients with mutation absent, holding tumor stage, age, and baseline disease burden constant

Interpreting Odds Ratios

  • odds ratio = 1 : no difference in odds

  • odds ratio > 1 : predictor is associated with higher odds of the outcome

  • odds ratio < 1 : predictor is associated with lower odds of the outcome

  • hypothesis test evaluates whether the coefficient equals zero on the log-odds scale

Logistic Regression Hypothesis Test

Coefficient of interest mutation status \(\beta_{\text{mutation}}\)

Hypothesis test evaluates whether mutation status is associated with treatment response after adjusting for other covariates [tumor stage, age, and baseline disease burden]

Null hypothesis:

\[ H_0: \beta_{mutation} = 0 \]

Alternative hypothesis:

\[ H_a: \beta_{mutation} \neq 0 \]

Interpretation:

  • under \(H_0\): mutation status has no association with the odds of response after adjustment for other variables
  • small p-value provides evidence that mutation status is associated with different odds of treatment response in the population

Summary

  • application of hypothesis testing within regression analysis
  • regression hypothesis tests evaluate whether a predictor is associated with an outcome
  • null hypothesis: the regression coefficient equals a specified value.
  • alternative hypothesis: the regression coefficient differs from the specified null value
  • regression test statistics measure how far an estimated coefficient is from its null value after accounting for sampling variability
  • p-values provide evidence for evaluating whether the observed coefficient is consistent with the null hypothesis
  • linear regression hypothesis tests are used when the outcome is continuous
  • logistic regression hypothesis tests are used when the outcome is binary
